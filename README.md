# Occupancy-Heatmap
<h1>Ανώνυμος Χάρτης Πληρότητας Σχολικών Χώρων</h1>

<hr>

<h2>Αναλυτική περιγραφή ιδέας</h2>

<p>
Το έργο αφορά την ανάπτυξη ενός <b>ανώνυμου συστήματος απεικόνισης παρουσίας</b>
σε σχολικό κτιριακό συγκρότημα.
</p>

<p>
Με χρήση αισθητήρων παρουσίας σε αίθουσες και κοινόχρηστους χώρους
και μικροελεγκτών <b>Arduino / ESP32</b>, καταγράφεται
<b>μόνο η ύπαρξη ή μη παρουσίας</b> σε κάθε χώρο,
χωρίς καταγραφή εικόνας, ήχου ή προσωπικών δεδομένων.
</p>

<p>
Η πληροφορία απεικονίζεται σε εφαρμογή <b>MIT App Inventor</b>
ως <b>χρωματικός χάρτης (heatmap)</b> κάτοψης του σχολείου,
επιτρέποντας άμεση εποπτεία της χρήσης των χώρων.
</p>

<p>
Το σύστημα έχει σχεδιαστεί με απόλυτο σεβασμό στην
<b>προστασία προσωπικών δεδομένων</b>, καθώς:
</p>

<ul>
  <li>Δεν καταγράφει ταυτότητα ατόμων</li>
  <li>Δεν αποθηκεύει εικόνα ή ήχο</li>
  <li>Δεν διατηρεί ιστορικό προσωπικών κινήσεων</li>
  <li>Επεξεργάζεται μόνο δυαδική πληροφορία παρουσίας (ΝΑΙ / ΟΧΙ)</li>
</ul>
